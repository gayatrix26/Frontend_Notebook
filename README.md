# Technical Notebook

A personal technical notebook built with Angular.

This application is used to organize and store my programming notes, code examples, and learning resources in one place. The goal is to create a structured notebook that makes it easy to revisit concepts and track my learning journey.

## Tech Stack

* Angular
* TypeScript
* HTML
* CSS

## Purpose

To maintain personal technical notes in a searchable and organized format instead of keeping them in PDFs or scattered documents.
